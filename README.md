# 동구 · 대구광역시

> 동구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-daegu-dong.github.io/**

동구는 대구광역시 8개 자치구·군 가운데 면적이 183㎢로 가장 넓은 자치구로, 대구의 북동쪽 관문이자 팔공산과 동대구역을 함께 품은 도시-산악 복합권역이다. 도심에는 경부고속철도가 정차하는 동대구역과 동대구복합환승센터가 자리해 영남권 광역교통의 허브 역할을 하고, 그 북쪽으로는 대구의 진산 팔공산이 병풍처럼 솟아 갓바위·동화사 같은 불교 성지를 거느린다. 대구국제공항이 동촌에 있어 하늘길까지 열려 있어, 철도·항공·고속버스가 한 구역 안에 모인 전국에서도 보기 드문 교통 결절지다. 신암·효목의 오래된 주거지와 안심·혁신동의 신도시, 봉무동 이시아폴리스 산업단지가 공존하며, 대구경북첨단의료복합단지와 대구혁신도시가 들어서 동남권 성장의 새 축을 이룬다. 후삼국 시대 공산 동수 전투의 무대이자 왕건과 신숭겸의 전설이 지명 곳곳에 새겨진 역사의 고장이기도 하다.
